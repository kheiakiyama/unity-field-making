# unity-field-making

Unity を使って土木の盛土作業の流れを見える化しました。

動作確認は[コチラ](https://dl.dropboxusercontent.com/u/76767589/unity-field-making/index.html)です。

## 操作方法

### 基本操作
WASD or 矢印キー：移動  
マウス：向き変更

### 説明
緑の犬をクリック：作業の開始・一時停止  
黄の犬をクリック：作業のスピードDown  
赤の犬をクリック：作業のスピードUp

<br>
作業員：オブジェです
